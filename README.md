# 📝 Quiz Management System (C++)

## 📌 Description
Console-based Quiz Management System in C++ with admin and user functionality. Supports question management, quiz taking, and result tracking.

## 🚀 Features
- Admin can add new questions
- Users can take quizzes
- Tracks score, percentage, and time taken
- Saves results in results.txt
- Saves questions in questions.txt

## 🛠️ Technologies Used
- C++ Programming
- File Handling (fstream)
- Vectors and Classes
- Time tracking

## ▶️ How to Run
1. Open quiz_system.cpp in your IDE (Dev-C++, VS Code, etc.)
2. Compile and run the program
3. For Admin: Add questions  
4. For User: Take the quiz

- quiz_system.cpp
